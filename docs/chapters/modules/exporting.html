13.4. Exporting Modules¶

We learned how to pull in useful code in the form of modules, but what if we have written clever code that we want to share? Fortunately, Node allows us to make our code available for other programs.

First, some basic points:

  1. Every Node.js file is treated as a module (also called a package).
  2. From a file, we can export a single function or a set of functions.
