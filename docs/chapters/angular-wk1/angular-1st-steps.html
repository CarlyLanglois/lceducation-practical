25.4. Angular First Steps¶

The goal here is to create a minimum working webpage template. It will serve as a jumping off point for your later exercises and studios, so be sure you are logged into your GitHub account.

25.4.1. Starting a New Project¶

Navigate to the StackBlitz homepage. Click “Start New App”, then select the “Angular (TypeScript)” option.

Examine the files created:

TODO: Add screenshot of file tree.
  1. The src folder holds all the files needed to make the template function.
  2. The app folder holds the instructions for displaying the smaller pieces of a webpage. We will modify these files soon.
  3. index.html is the highest level for displaying content. Anything added to this page will appear on every page within a website.
  4. main.ts imports the core methods required to make everything work. It also imports the app modules, which control the appearance of a webpage.
  5. styles.css holds the global style settings for the entire website.
