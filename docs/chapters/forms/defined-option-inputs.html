17.7. Defined Option Inputs¶

17.7.1. Defined Option Inputs¶

Type Syntax Description Demo
range <input type="range" name="volume"/> A slider that allows the user to input a numeric value within the given range.
select <select name="weather"><option>clear</option><option>cloudy</option></select> A drop down menu that allows selection of one option. Requires options to be in <option> tags.
