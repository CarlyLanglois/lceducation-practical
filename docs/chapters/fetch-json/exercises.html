24.5. Exercises¶

24.5.1. Valid JSON¶

Which of the following three code snippets is correct JSON? Why are the other two options incorrect?

{
    type: "dog",
    name: "Bernie",
    age: 3
}

{
    "type": "dog",
    "name": "Bernie",
    "age": 3
}

{
    "type": 'dog',
    "name": 'Bernie',
    "age": 3
}

Which of the following three code snippets is correct JSON? Why are the other two options incorrect?

{
    "animals": [
        {
            "type": "dog",
            "name": "Bernie",
            "age": 3
        },
        {
            "type": "cat",
            "name": "Draco",
            "age": 2
        }
    ]
}

{
    [
       {
            "type": "dog",

            "age": 3
        },
        {

    ]
}
