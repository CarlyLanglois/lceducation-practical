24.4. Fetching Data¶

Now that we know what an API, let’s use one to update a web page. Let’s use a weather API to add weather data to a webpage. The URL for this special LaunchCode weather API is https:://weather.education.launchcode.org.

Example JSON returned from weather API.

{
   "temp": 67,
   "windSpeed": 5,
   "tempMin": 50,
   "tempMax": 71,
   "status": "Sunny",
   "chanceOfPrecipitation": 20,
   "zipcode": 63108
}

We can see that this API returns useful information like temp and windSpeed. Our goal is to add that data to a Launch Status website.
