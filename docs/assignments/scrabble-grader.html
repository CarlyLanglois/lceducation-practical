Assignment #2: Scrabble Grader¶

To see how far you skills have come, we are going to give you a mission to update our super important Scrabble Grader program. What, did you think you were going to work on Mars rover code already?

Requirements¶

  1. Create an oldScoreKey object that contains the previous scoring system.
  2. Create a transform function which will return a newScoreKey object.
  3. Create scoringAlgorithms array to hold three scoring objects, which include the scoring functions.
  4. Create a initialPrompt function that asks user what score algorithm to use.
  5. Create a runProgram function to contain the starting point for your running program.

const oldScoreKey = {
   1: ['A', 'E', 'I', 'O', 'U', 'L', 'N', 'R', 'S', 'T'],
   2: ['D', 'G'],
   3: ['B', 'C', 'M', 'P'],
   4: ['F', 'H', 'V', 'W', 'Y'],
   5: ['K'],
   8: ['J', 'X'],
   10: ['Q', 'Z']
};

Write a transform function that takes in as a parameter the oldScoreKey object. transform(oldScoreKey) will return an object with the letters as keys. The keys in the newScoreKey should be lowercase letters.

Example of new key storage

  • a is worth 1
  • j is worth 8
  • k is worth 10

ALL scoring algorithms should be case insensitive, they will act the same for lower and upper case
