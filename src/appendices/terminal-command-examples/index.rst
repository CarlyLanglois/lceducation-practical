Terminal Commands
=================


=========================================   ================================    ===========================    
``.``                                       current directory                   :ref:`terminal-current-dir`         
``..``                                      move to parent directory            :ref:`terminal-parent-dir`
``pwd``                                     print working directory             :ref:`terminal-pwd` 
``ls``                                      list items in current directory     details
``cd <path_name>``                          change directory                    details
``mkdir <new_directory_name>``              make new directory                  details
``rm <item_to_remove>``  (``-r``)           remove item (remove directory)      details
``cp <source_path> <target_path>``          copy item                           details
``mv <item_to_move> <target_path>``         move item                           details
``touch <new_file_name>``                   make new file                       details
*ctrl + c*                                  exit running program                details
``clear``                                   clean screen                        details
``man <command>``                           look up command manual entry        details
``q``                                       exit the manual                     details
